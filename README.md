# PrintGuard – 3D Printer Farm Monitoring

## Overview

PrintGuard is a Fog and Edge Computing project that monitors a 3D printer farm using simulated IoT sensors.

The system demonstrates local fog processing before transmitting processed data to cloud services.

## Features

- Multi-printer sensor simulation
- MQTT communication
- Fog-based anomaly detection
- Health score calculation
- Recommendation engine
- Aggregation
- Batch processing
- AWS cloud integration (in progress)

## Technologies

- Python
- MQTT (Mosquitto)
- Paho MQTT
- AWS API Gateway
- AWS Lambda
- Amazon DynamoDB
- GitHub Actions
